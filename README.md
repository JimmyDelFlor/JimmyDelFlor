## 🚀 Proyectos destacados

### 🏠 SmartHome IoT System
Sistema domótico con arquitectura de 4 capas: interfaz React → backend Node.js → modelo de lenguaje LLaMA 3.1 vía Ollama → microcontroladores ESP32/Arduino con sensores.
Incluye pipeline de voz con **Whisper STT** y Google Drive como cola de mensajes asíncrona.
> `React` `Node.js` `LLaMA 3.1` `Ollama` `ESP32` `Arduino` `Whisper` `Socket.IO`

---

### 🌿 Detección de caña con Visión Artificial (CNN)
Modelo de detección entrenado con **TensorFlow/Keras** en Google Colab, capaz de identificar cañas de azúcar y sugerir medidas de acción. Procesamiento de imágenes con **OpenCV**.
> `Python` `TensorFlow` `Keras` `OpenCV` `Google Colab` `Jupyter Notebook`

---

### 📊 Dashboard Power BI — Cafetería Misky
Dashboard de análisis de ventas con modelado dimensional en estrella y consultas SQL. Redujo el tiempo de generación de reportes en **~80%**.
> `Power BI` `SQL` `Modelado dimensional` `Star schema`
